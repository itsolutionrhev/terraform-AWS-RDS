# terraform-AWS-RDS
>Amazon RDS is the Relational Database Service offered as a web service by Amazon. 

>We can Install, setup and configure operational databasse on AWS Cloud.

>AWS RDS is a leading RDBMS managed database service. 

>AWS RDS provide stable and reliable database service without install any server platform we need to just choose AWS RDS as service with reliable CPU,Memory and other initial configuration and use it.

> AWS RDS manage Auto Backup, Software paching, and Auto High Availablity and recovery of RDBMS.
 
 >AWS RDS service provide many relational database like EX...
 
	>MySQL - Support versions for MySQL 5.5 to 5.7 and more..

	>MariaDB – Support versions for MariaDB from 10.0 to 10.2 and more..

	>Oracle – Supports version 11g and 12c and more.

	>Microsoft SQL Server – Supports version 200t to 2017 and more..

	>PostgreSQL – Supports version 9 to 11 and more..

	>Amazon Aurora – Amazon’s own RDBMS. 

> AWS RDS support Auto scalling of stporage capacity for DB workload with "0" Downtime.

####### AWS-RDS Setup Requirements ######

1)Subnet-Group : Will provide subnet (Network range) for database Instance.

2)Parameters-Group : will contains RDBMS settings.

3)Security-Group : Will allows incoming traffic for RDBMS.

> RDS Instance requirement
	: t2.micro
	: 2GB RAM
	: 1 CPU
